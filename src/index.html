Open source

  • land-registry-elements

    Component library for use when creating Land Registry services. Each component comes with the necessary CSS & JS as well as Jinja templates for the calling application to implement it.

  • govuk-elements-jinja-macros

    Jinja macros for use when creating GOV.UK services in Python/Flask. Integrates neatly with Flask-WTF (WTForms) to provide GOV.UK compliant form element markup, including validation and errors.

  • node-wpautop

    Node.JS port of WordPress' wpautop() function. Useful for feeding WordPress content into a Node.JS based site.

  • conduct.js

    conduct.js allows JavaScript to respond to media queries by executing callbacks as breakpoints are matched and unmatched. It is inspired by enquire.js and arose from the need to gain additional control over the order of execution of breakpoint callbacks.
